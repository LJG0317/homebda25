# 📊 Data Analyst Portfolio

**신입 데이터 분석가 | Python · SQL · Excel · Tableau**

## 👤 About Me

데이터를 통해 **비즈니스 문제를 정의하고, 의사결정에 기여하는 신입 데이터 분석가**입니다.
Python과 SQL을 활용한 데이터 분석 경험과 Tableau를 통한 시각화 및 인사이트 전달에 강점이 있습니다.
단순 분석에 그치지 않고, **"이 결과로 무엇을 할 수 있는가"**를 항상 고민하며 프로젝트를 수행했습니다.

* 데이터 전처리 → 분석 → 시각화 → 리포팅까지 End-to-End 경험
* 실무 시나리오 기반 프로젝트 중심 학습
* 분석 결과를 비즈니스 언어로 설명하는 것을 목표로 함
